# SVS School Website

A single-page website for Sri Vijaya Sai School, Bengaluru — an English-medium 
school serving 700+ students from Nursery to Grade 10, established in 2000.

**Live:** https://svs-school.netlify.app

## Design highlights
- Kid-friendly but authoritative tone for parents
- Responsive across desktop, tablet, and phone
- WCAG AA accessibility throughout
- Paper airplane cursor + sparkle trail on desktop
- Custom form submission via mailto

## Tech
- Single HTML file, no build step
- Vanilla CSS and JavaScript
- Google Fonts (Lilita One + DM Sans)
- Google Maps embed for directions

Built iteratively with Claude.
